## 食用教程

```bash
git clone https://github.com/wibus-wee/Typecho-Plugin-IuckyMagic.git
```
将文件夹改名为IuckyMagic并上传至 typecho 的插件usr/plugins目录。
登录后台启用IuckyMagic插件即可正确食用。
